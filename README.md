# Aero-Web

A responsive website developed for an aeromodelling/aerospace club to showcase its activities, projects, events, achievements, and team. The website is designed to provide visitors with an engaging experience while maintaining a clean and professional interface.

## Features

- Responsive design for desktop, tablet, and mobile devices
- Modern and interactive user interface
- Homepage with club introduction
- Projects showcase
- Events section
- Team members page
- Gallery of activities
- Contact section
- Smooth navigation and animations
- Fast loading and optimized performance

## Installation

Clone the repository:

```bash
git clone https://github.com/shailendra241281/Aero-Web.git
```

Move into the project directory:

```bash
cd Aero-Web
```

If it is a static website, simply open `index.html` in your browser.

If it uses npm:

```bash
npm install
npm run dev
```
